# helloWorld
DBOS Hello World function (TypeScript)

## Add neccesary things to local files
* git clone https://github.com/spiritfx69/helloWorld
* cd helloWorld
* npm init -y
* npm install --save-dev typescript
* npx tsc -v                                                 // check TS version
* npm run build
* npm start
